# 上瘾机制优化报告 - 让用户持续分泌多巴胺

> 实施日期：2026-06-26
> 核心目标：解决"无法让用户上瘾"的问题，植入持续的多巴胺循环

---

## 问题诊断

### 原有问题

1. ❌ **互动单调**：点击/喂食/玩游戏 → 数值+1，缺少惊喜和变化
2. ❌ **游戏无参与感**：激光只是看动画，不是真正"玩"
3. ❌ **成长路径单一**：好感/经验只有一种获取方式
4. ❌ **缺少收集目标**：没有解锁、成就、稀有物
5. ❌ **行走倒退**：facing 逻辑问题导致视觉错误
6. ❌ **过度行走**：待机权重失衡，影响用户视觉

### 核心症结

**缺少"上瘾循环"** —— 用户没有持续的动机和即时反馈来分泌多巴胺。

---

## 解决方案：植入五层多巴胺循环

### ✅ 第1层：变量奖励系统（即时多巴胺）

**心理学原理**：Skinner 的变量奖励 —— 不确定的奖励比固定奖励更上瘾。

**实施内容**：

1. **奖励随机化**（±20%波动）
   - 每次喂食/摸头/玩游戏，奖励在基础值上下波动
   - 消除"可预测性"，每次都有小惊喜

2. **暴击系统**（稀有高额奖励）
   - 喂食：10%概率触发，双倍奖励
   - 游戏：15%概率触发，双倍奖励
   - 摸头：5%概率触发，**三倍奖励**（稀有度高，惊喜强）

3. **多样化反馈**
   - 单击猫：6种随机反应（话术+是否弹跳）
   - 喂食：4种随机台词
   - 摸头：4种随机台词
   - 暴击时专属特效：粒子爆发+专属台词（"哇！超级好吃！！！💖"）

**效果**：用户每次互动都期待"这次会不会暴击？会说什么？"，形成**即时反馈循环**。

---

### ✅ 第2层：连击系统（节奏多巴胺）

**心理学原理**：节奏感 + 目标梯度效应 —— 越接近目标，投入度越高。

**实施内容**：

1. **连击计数**：10秒内连续互动计数
2. **连击加成**：每连击1次，奖励+5%（最高30%，封顶6连击）
3. **视觉反馈**：3连击以上显示"⚡ X连击！"
4. **节奏感**：鼓励用户"再来一次！再来一次！"

**代码示例**：
```javascript
// 10秒内连续互动计数
const COMBO_WINDOW = 10000;
const timeSinceLastInteraction = now - (pet.lastInteractionAt || 0);
let comboCount = timeSinceLastInteraction < COMBO_WINDOW ? pet.comboCount + 1 : 1;

// 连击加成
const comboBonus = Math.min(comboCount - 1, 6) * 0.05;
const multiplier = (isCritical ? 2 : 1) * (1 + comboBonus);
```

**效果**：制造"一气呵成"的爽感，让用户停不下来。

---

### ✅ 第3层：参与式游戏（操作多巴胺）

**心理学原理**：自我效能感 —— 用户需要感受到"我在控制，我在影响"。

**原问题**：激光笔是自动播放动画，用户只能看，无参与感。

**重构方案**：

1. **鼠标控制光点**：光点跟随鼠标移动，用户完全控制
2. **追逐机制**：猫以可变速度追光点（远→快，近→慢）
3. **抓到判定**：距离<25px算抓到，即时弹跳+话术（"抓到了！"）
4. **表现影响奖励**：
   - 抓到≥5次："好玩！😸" + 高额奖励
   - 抓到2-4次："还行吧" + 正常奖励
   - 抓到<2次："哼，一次都没抓到…" + 低奖励（不惩罚但有反馈）
5. **时长延长**：从15秒增加到20秒，给足互动时间

**代码变化**：
```javascript
// 原：点击移动光点（被动观看）
window.addEventListener('mousedown', clickHandler);

// 新：鼠标移动控制光点（主动参与）
window.addEventListener('mousemove', moveHandler);

// 表现影响奖励
const performanceMultiplier = catchCount ? Math.min(1 + catchCount * 0.1, 2) : 1;
```

**效果**：从"看动画"变成"玩游戏"，用户获得掌控感和即时反馈。

---

### ✅ 第4层：成就系统（长期多巴胺）

**心理学原理**：收集欲 + 完成感 —— 人天生喜欢收集和"打勾"。

**实施内容**：

**成就列表（12个）**：
1. 🍖 **第一次投喂** —— 首次喂食
2. ⚡ **5连击** —— 连续互动5次
3. ⚡⚡ **10连击** —— 连续互动10次
4. 💥 **幸运之星** —— 触发一次暴击
5. 🌟 **初露锋芒** —— 达到5级
6. 🌟🌟 **小有成就** —— 达到10级
7. 💕 **亲密无间** —— 好感度达到50
8. 💕💕 **心心相印** —— 好感度100
9. 🎮 **热情玩家** —— 累计互动100次
10. 🎮🎮 **资深玩家** —— 累计互动500次
11. 🌙 **夜猫子** —— 深夜（23点后/6点前）还在陪伴
12. ⚡ **激光大师** —— 一局游戏抓到10次光点

**解锁反馈**：
- 气泡显示："✨成就解锁✨\n[名称]\n[描述]"
- 庆祝特效（粒子爆发）
- 多个成就依次解锁（间隔4.5秒避免重叠）

**持久化**：
```javascript
achievements: {
  firstFeed: false,
  combo5: false,
  // ... 12个成就标记
}
```

**效果**：给用户长期追求的目标，"我还差几个成就？"驱动回访。

---

### ✅ 第5层：视觉优化（体验多巴胺）

**问题1：行走倒退**

**原因**：`advanceWalk()` 每帧重新计算朝向，导致抖动。

**修复**：
```javascript
// startWalk 时固定朝向
this.walkDir = this.walkTarget > pet.x ? 1 : -1;
pet.setFacing(this.walkDir);

// advanceWalk 使用固定朝向
const dir = this.walkDir || 1; // 不再每帧重新计算
```

**问题2：过度行走**

**原因**：行走概率55%太高，待机时间过短。

**修复**：
- 行走概率从55%降到**25%**
- 待机时间从4-10秒延长到**5-12秒**
- 增加"安静待机"分支（不弹气泡，只是静静呆着）

**效果**：猫咪不再疯狂乱跑，更符合真实猫的懒散气质。

## 技术实现细节

### 改动文件清单

1. ✏️ `src/shared/constants.js` — 新增 comboCount, lastInteractionAt, totalInteractions, skinColor, achievements
2. ✏️ `src/main/store.js` — 新增成就管理（getAchievements, unlockAchievement）
3. ✏️ `src/main/ipc.js` — 变量奖励、暴击、连击、成就检查
4. ✏️ `src/renderer/main.js` — 成就通知、连击显示、随机反馈
5. ✏️ `src/renderer/pet/BehaviorTree.js` — 修复行走、优化待机权重
6. ✏️ `src/renderer/games/LaserChase.js` — 重构为鼠标控制的真正游戏
7. ✏️ `src/preload.js` — 支持 catchCount 参数传递

### 向后兼容性

- 所有新增字段通过 `DEFAULT_DATA` 定义
- `store.js` 的 `deepMerge` 自动兼容旧存档
- 旧版本升级后成就从零开始计数

### 性能考虑

- 成就检查只在互动时触发，不占用主循环
- 连击计时器用时间差计算，无需额外定时器
- 成就通知异步延迟，避免阻塞主流程

## 总结

### ✅ 已解决的核心问题

1. ✅ 互动单调 → **变量奖励 + 暴击 + 随机反馈**
2. ✅ 游戏无参与感 → **鼠标控制 + 表现影响奖励**
3. ✅ 成长路径单一 → **连击加成 + 游戏表现加成**
4. ✅ 缺少收集目标 → **12个成就 + 解锁通知**
5. ✅ 行走倒退 → **固定朝向不抖动**
6. ✅ 过度行走 → **降低频率 + 延长待机**

### 🎯 植入的多巴胺循环

- **即时多巴胺**（1-3秒）：变量奖励、暴击、随机反馈
- **节奏多巴胺**（10秒-5分钟）：连击系统、游戏参与
- **长期多巴胺**（数天-数周）：成就收集、等级进度

### 💡 核心优势

1. **零新增美术**：全部使用现有资源
2. **心理学驱动**：每个设计都有理论支撑
3. **数据可调**：概率/倍数/阈值都可配置
4. **向后兼容**：旧存档无缝升级
