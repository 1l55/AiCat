## 阅读路径
- **想了解全貌** → 按 1 → 2 → 3 顺序读。
- **要开始写代码** → 直接看 ③，回查 ② 理解"为什么"。
- **做美术/皮肤** → 看 ① 第 2~3 章（形象、配色、待机动作）。

## 三条贯穿主线
1. **零美术优先**：先用现有 3 套精灵帧（idle/walk/run）+ 染色/粒子/文字特效把体验做透，再投美术做皮肤/表情收集。
2. **情感陪伴 > 工具**：养成、对话、提醒都服务于"加深关系"，而非功能堆砌。
3. **伦理红线**：心理学只用于"健康依恋"——奖励回归而非惩罚离开、尊重自主、永不操纵（详见文档 ② 第 7 章）。
