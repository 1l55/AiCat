# 快速测试清单

## 🎯 核心上瘾机制测试

### 1. 变量奖励 + 暴击系统
- [ ] **喂食10次**，观察是否有暴击（10%概率，预期1次）
  - 暴击时应显示："哇！超级好吃！！！💖" + 粒子爆发
- [ ] **摸头20次**，观察是否有暴击（5%概率，预期1次）
  - 暴击时应显示："咕噜咕噜～超级舒服！！💕" + 粒子爆发
- [ ] 观察每次反馈是否不同（随机话术）

### 2. 连击系统
- [ ] **快速连续喂食3次**（间隔<10秒）
  - 应显示"⚡ 3连击！"
- [ ] **继续喂食到5次**
  - 应显示"⚡ 5连击！"
- [ ] **等待11秒后再喂食**
  - 连击应重置为1

### 3. 激光游戏重构
- [ ] **右键菜单 → 玩耍**
  - 开场应提示："光点！快控制光点逗我玩！"
- [ ] **移动鼠标**
  - 光点应跟随鼠标
  - 猫应追逐光点
- [ ] **让猫抓到光点**（光点停住，等猫靠近<25px）
  - 猫应弹跳 + 显示"抓到了！"/"嘿嘿～"等
- [ ] **游戏结束**（20秒）
  - 根据抓到次数显示不同反馈

### 4. 成就系统
- [ ] **首次喂食**
  - 应解锁"🍖 第一次投喂"成就
  - 显示成就通知 + 粒子特效
- [ ] **触发一次暴击**
  - 应解锁"💥 幸运之星"
- [ ] **5连击**
  - 应解锁"⚡ 5连击"
- [ ] **深夜测试**（23点后或6点前）
  - 互动应解锁"🌙 夜猫子"

### 5. 视觉修复
- [ ] **观察猫咪行走**
  - 不应倒退
  - 朝向应与移动方向一致
- [ ] **观察待机频率**
  - 不应过度行走
  - 应有较长时间静静待机

## 🎬 快速演示流程（5分钟）

1. **启动应用** → 观察重逢欢迎（如果是重启）
2. **连续喂食5次** → 看连击显示 + 期待暴击
3. **双击摸头10次** → 看随机反馈 + 期待稀有暴击
4. **玩激光游戏** → 鼠标控制，尝试抓到10次
5. **查看成就** → 确认已解锁的成就数量
6. **观察待机** → 猫不应疯狂乱跑
7. **打开状态面板** → 看连击次数、总互动次数

## 🔧 如遇问题

### 暴击不触发？
- 概率：喂食10%、游戏15%、摸头5%
- 多试几次，摸头需要试20+次

### 连击不显示？
- 确保间隔<10秒
- 需要≥3连击才显示

### 激光光点不跟随鼠标？
- 确保窗口可交互（游戏开始时自动设置）
- 尝试重新启动游戏

### 成就不解锁？
- 检查控制台是否有错误
- 确认 petdata.json 中 achievements 字段存在
